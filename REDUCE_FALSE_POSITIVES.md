# Снижение Вероятности для Легитных Игроков

## Проблема
Легитные игроки показывают 30-50% вместо 10-20%

## Решение

### 1. Улучшен ViolationTracker
- Порог для легитных повышен с 0.1 до 0.3
- Максимальное снижение увеличено с -15% до -25%
- Добавлена прогрессивная формула снижения

### 2. Улучшена Модель (train_model.py)
- n_estimators=300 (было 100)
- max_depth=15 (было 10)
- Добавлены min_samples_split=10, min_samples_leaf=5
- class_weight='balanced' для балансировки классов

### 3. Улучшен API Сервер (api_server.py)
- Новая консервативная агрегация
- 40% avg + 30% median + 20% p75 + 10% p90

## Как Применить

### Шаг 1: Переобучить модель
```cmd
python train_model.py
```

### Шаг 2: Перезапустить API
```cmd
restart_api.bat
```

### Шаг 3: Обновить плагин
```cmd
gradlew build
```
Скопировать `build/libs/untitled-1.0.jar` в `plugins/`

### Шаг 4: Перезапустить сервер

## Ожидаемые Результаты
- Легит: 10-20% (было 30-50%)
- Читер: 85-95% (было 68-75%)
