# Как исправить ложные срабатывания (легит игроки до 0.50)

## Быстрое решение (3 шага)

### Шаг 1: Переобучите модель
```bash
python train_model.py
```

Или просто запустите `retrain.bat`

### Шаг 2: Перезапустите API сервер

1. Остановите текущий API (Ctrl+C)
2. Запустите снова:
```bash
python api_server.py
```

### Шаг 3: Проверьте результаты

Зайдите на сервер и проверьте легитного игрока:
```
/smartai prob <ник_легитного_игрока>
```

Вероятность должна быть **< 0.30** для легитных игроков.

## Если все еще есть проблемы

### 1. Соберите больше легитных данных
```
/smartai start <легитный_игрок> legit
```
Играйте 5-10 минут в PvP, затем:
```
/smartai stop <легитный_игрок>
```

### 2. Повысьте пороги в config.yml
```yaml
alerts:
  threshold: 0.80  # было 0.75

violation:
  threshold: 50    # было 40
```

### 3. Переобучите модель снова
```bash
python train_model.py
```
