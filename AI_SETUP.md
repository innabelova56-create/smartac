# Где найти AI? Создай сам!

## Быстрый старт (5 шагов)

### 1. Собери данные на сервере
```bash
# Запусти Minecraft сервер с плагином
/smartai start legit
# Играй в PvP 100+ боев
/smartai stop

/smartai start cheat  
# Включи KillAura/AimBot, играй 100+ боев
/smartai stop
```

Данные сохранятся в `plugins/SmartAI/datasets/`

### 2. Установи Python библиотеки
```bash
# Базовый вариант (Random Forest)
pip install pandas scikit-learn numpy joblib flask

# Продвинутый вариант (LSTM нейросеть)
pip install pandas scikit-learn numpy joblib flask tensorflow
```

### 3. Обучи модель

**Вариант A: Random Forest (быстро, просто)**
```bash
python train_model.py
```
Создаст: `smartai_model.pkl`

**Вариант B: LSTM (лучше качество)**
```bash
python train_lstm_model.py
```
Создаст: `smartai_lstm_model.h5` и `smartai_scaler.pkl`

### 4. Запусти API сервер

**Для Random Forest:**
```bash
python api_server.py
```

**Для LSTM:**
```bash
python api_server_lstm.py
```

Сервер запустится на `http://localhost:5000`

### 5. Проверь работу
```bash
# Простая проверка
curl http://localhost:5000/health

# Или запусти тестовый скрипт
python test_api.py
```

Должен вернуть:
```json
{
  "status": "ok",
  "model_loaded": true
}
```

## Сколько нужно данных?

| Качество | Легит боев | Читер боев | Точность |
|----------|-----------|-----------|----------|
| Минимум  | 50        | 50        | ~70%     |
| Хорошо   | 200       | 200       | ~85%     |
| Отлично  | 1000+     | 1000+     | ~95%+    |

## Какую модель выбрать?

### Random Forest (train_model.py)
**Плюсы:**
- Быстрое обучение (1-5 минут)
- Не требует GPU
- Простая установка
- Хорошо работает на малых данных

**Минусы:**
- Точность ниже (~85-90%)
- Не учитывает временные паттерны

**Используй если:** мало данных или нет GPU

### LSTM (train_lstm_model.py)
**Плюсы:**
- Высокая точность (~95%+)
- Учитывает последовательность движений
- Лучше детектирует сложные читы

**Минусы:**
- Долгое обучение (10-30 минут)
- Требует больше данных
- Нужен TensorFlow

**Используй если:** много данных и нужна максимальная точность

## Troubleshooting

### "No CSV files found"
- Проверь путь к папке с данными
- Убедись что собрал данные через `/smartai start/stop`

### "Model not loaded"
- Проверь что файл модели существует
- Переобучи модель

### "Not enough data"
- Собери минимум 50 боев каждого типа
- Чем больше, тем лучше

### Низкая точность
- Собери больше данных
- Используй LSTM вместо Random Forest
- Проверь баланс классов (легит vs читер)
