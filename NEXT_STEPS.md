# Что делать дальше

## 1. Переобучите модель с улучшенными параметрами

```bash
python train_model.py
```

Или запустите `retrain.bat`

Это улучшит точность и снизит вероятности для легитных игроков с 0.50 до 0.20-0.30.

---

## 2. Перезапустите API сервер

1. Остановите текущий процесс (Ctrl+C в терминале где запущен api_server.py)
2. Запустите снова:
```bash
python api_server.py
```

---

## 3. Пересоберите плагин

```bash
./gradlew build
```

Новый jar будет в `build/libs/`

---

## 4. Установите плагин на сервер

1. Скопируйте `build/libs/SmartAI-1.0.jar` в папку `plugins/` вашего сервера
2. Перезапустите сервер или используйте `/reload confirm`

---

## 5. Проверьте новые возможности

### Новый конфиг
- Откройте `plugins/SmartAI/config.yml`
- Настройте пороги, пенальти, GUI
- Настройте `sample-interval` (рекомендуется 10)

### Новые команды
```
/smartai start <player|global> <cheat|legit> [name]  - Сбор данных
/smartai stop <player|global>                        - Остановить сбор
/smartai alerts                                      - Переключить алерты
/smartai prob <player>                               - Отслеживание вероятности
/smartai datastatus                                  - Статус сбора данных
/smartai reload                                      - Перезагрузить конфиг
/smartai suspects                                    - GUI (в разработке)
/smartai punish <player>                             - Наказание (в разработке)
```

### Messages.yml
- Все сообщения теперь в `plugins/SmartAI/messages.yml`
- Можно настроить цвета, форматы, тексты

### Голограммы
Над подозрительными игроками будут голограммы с последними 5 ударами и средним значением.
Теперь используют систему пассажиров - идеально следуют за игроком без телепортации.

---

## 6. Соберите больше данных (опционально)

Для лучшей точности соберите больше данных:

### Легитные игроки:
```
/smartai start <player> legit
```
Играйте 5-10 минут, затем:
```
/smartai stop <player>
```

### Читеры:
```
/smartai start <player> cheat
```
Пусть используют читы 5-10 минут, затем:
```
/smartai stop <player>
```

После сбора новых данных - переобучите модель снова (шаг 1).

## Ожидаемые результаты

После переобучения:
- **Легитные игроки**: 0.10 - 0.30 (зеленый)
- **Подозрительные**: 0.60 - 0.80 (оранжевый)  
- **Читеры**: 0.80+ (красный)

Точность модели должна быть >90% на тестовой выборке.
