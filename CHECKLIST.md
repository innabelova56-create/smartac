# ✅ Чеклист запуска SmartAI

## Этап 1: Установка плагина

- [ ] Java 21 установлена
- [ ] Paper 1.21.4 сервер запущен
- [ ] Скомпилирован плагин: `./gradlew build`
- [ ] JAR скопирован в `plugins/`
- [ ] Сервер перезапущен
- [ ] Плагин загрузился без ошибок (проверь логи)
- [ ] Команда `/smartai` работает

## Этап 2: Сбор данных

### Легитные данные
- [ ] Запущен сбор: `/smartai start legit`
- [ ] Проведено минимум 50 боев
- [ ] Разные игроки (если возможно)
- [ ] Разное оружие (меч, топор)
- [ ] Остановлен сбор: `/smartai stop`
- [ ] CSV файл создан в `plugins/SmartAI/datasets/`

### Данные читеров
- [ ] Установлен чит-клиент (Wurst, LiquidBounce и т.д.)
- [ ] Запущен сбор: `/smartai start cheat`
- [ ] Включен KillAura/AimBot
- [ ] Проведено минимум 50 боев
- [ ] Разные настройки читов (если возможно)
- [ ] Остановлен сбор: `/smartai stop`
- [ ] CSV файл создан

### Проверка данных
- [ ] В папке `datasets/` есть файлы `LEGIT_*.csv`
- [ ] В папке `datasets/` есть файлы `CHEAT_*.csv`
- [ ] Файлы не пустые (больше 100 строк каждый)
- [ ] Открыл CSV в Excel/LibreOffice - данные выглядят нормально

## Этап 3: Подготовка Python окружения

- [ ] Python 3.8+ установлен: `python --version`
- [ ] pip установлен: `pip --version`
- [ ] Установлены зависимости: `pip install -r requirements.txt`
- [ ] Нет ошибок при установке

## Этап 4: Обучение модели

### Вариант A: Random Forest
- [ ] Скопированы CSV файлы в нужную папку
- [ ] Запущено обучение: `python train_model.py`
- [ ] Обучение завершилось без ошибок
- [ ] Создан файл `smartai_model.pkl`
- [ ] Точность модели > 70%

### Вариант B: LSTM (опционально)
- [ ] TensorFlow установлен: `pip install tensorflow`
- [ ] Запущено обучение: `python train_lstm_model.py`
- [ ] Обучение завершилось без ошибок
- [ ] Созданы файлы `smartai_lstm_model.h5` и `smartai_scaler.pkl`
- [ ] Точность модели > 80%

## Этап 5: Запуск API сервера

- [ ] Выбран тип сервера (Random Forest или LSTM)
- [ ] Запущен сервер: `python api_server.py` или `python api_server_lstm.py`
- [ ] Сервер запустился без ошибок
- [ ] Порт 5000 свободен
- [ ] Health check работает: `curl http://localhost:5000/health`
- [ ] Тест работает: `python test_api.py`

- [ ] Настроен `config.yml` с URL API
- [ ] Плагин отправляет запросы к API

## Troubleshooting

### Плагин не загружается
- [ ] Проверил версию Java: `java -version` (должна быть 21)
- [ ] Проверил версию Paper (должна быть 1.21.4)
- [ ] Проверил логи сервера на ошибки
- [ ] PacketEvents загружается корректно

### Данные не сохраняются
- [ ] Проверил права на папку `plugins/SmartAI/datasets/`
- [ ] Проверил что команда `/smartai stop` выполнена
- [ ] Проверил логи на ошибки IOException

### Модель не обучается
- [ ] Проверил что CSV файлы в правильной папке
- [ ] Проверил что есть оба типа данных (LEGIT и CHEAT)
- [ ] Проверил что файлы не пустые
- [ ] Проверил что все зависимости установлены

### API не работает
- [ ] Проверил что модель обучена и файл существует
- [ ] Проверил что порт 5000 не занят: `netstat -an | grep 5000`
- [ ] Проверил firewall
- [ ] Попробовал перезапустить сервер

### Низкая точность
- [ ] Собрал больше данных (минимум 200 боев каждого типа)
- [ ] Проверил баланс классов (примерно 50/50)
- [ ] Попробовал LSTM вместо Random Forest
- [ ] Проверил качество данных (нет дубликатов, ошибок)

## Полезные команды

```bash
# Компиляция плагина
./gradlew build

# Установка Python зависимостей
pip install -r requirements.txt

# Обучение Random Forest
python train_model.py

# Обучение LSTM
python train_lstm_model.py

# Запуск API (Random Forest)
python api_server.py

# Запуск API (LSTM)
python api_server_lstm.py

# Тестирование API
python test_api.py

# Проверка health
curl http://localhost:5000/health

# Просмотр логов сервера
tail -f logs/latest.log
```
