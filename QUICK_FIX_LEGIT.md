# Быстрое Исправление: Легит 10-20% вместо 30-50%

## Что Изменено

✅ ViolationTracker - агрессивное снижение для легитных (до -25%)
✅ train_model.py - лучшие параметры модели (300 деревьев, глубина 15)
✅ api_server.py - консервативная агрегация (меньше ложных срабатываний)

## Применить Изменения

```cmd
REM 1. Переобучить модель
python train_model.py

REM 2. Перезапустить API
restart_api.bat

REM 3. Пересобрать плагин
gradlew build

REM 4. Скопировать JAR в plugins/
copy build\libs\untitled-1.0.jar plugins\

REM 5. Перезапустить сервер
```
