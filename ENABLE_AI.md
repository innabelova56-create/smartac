# Как включить AI проверку

## Проблема
Вы видите в логах:
```
[SmartAI] [AI] AI проверка отключена в конфиге
```

## Решение

### Шаг 1: Запустите API сервер

Откройте новый терминал и запустите:
```bash
cd C:\Users\Admin\Desktop\untitled
python api_server.py
```

Должно показать:
```
==================================================
Starting SmartAI API Server...
==================================================
Модель загружена успешно
 * Running on http://0.0.0.0:5000
```

**Оставьте этот терминал открытым!**

---

### Шаг 2: Включите AI в конфиге

1. Остановите сервер: `/stop`

2. Откройте файл конфига:
```
[путь_к_серверу]/plugins/SmartAI/config.yml
```

3. Найдите секцию `detection:` и измените:
```yaml
detection:
  enabled: true    # ← Измените на true
  endpoint: "http://localhost:5000"
  sample-size: 40
  sample-interval: 10
```

Если такой секции нет, замените весь файл на новый конфиг из `src/main/resources/config.yml`

4. Сохраните файл

---

### Шаг 3: Запустите сервер снова

Запустите сервер и проверьте логи. Теперь должно быть:
```
[SmartAI] [Голограммы] Включены!
[SmartAI] [AI] Подключено к API: http://localhost:5000
[SmartAI] [AI] AI проверка включена!
[SmartAI] [AI] Sample size: 40 ticks
[SmartAI] SmartAI включен! Используйте /smartai для сбора данных.
[SmartAI] AI детекция активна! Админы получат алерты при подозрительной активности.
```

---

## Быстрая проверка

### 1. Проверьте что API работает

В браузере откройте:
```
http://localhost:5000/health
```

Должно показать:
```json
{
  "status": "ok",
  "model_loaded": true
}
```

### 2. Проверьте в игре

Зайдите на сервер и введите:
```
/smartai datastatus
```

Должно работать без ошибок.

---

## Если API не запускается

### Ошибка: "Address already in use"

Порт 5000 уже занят. Найдите процесс:
```bash
netstat -ano | findstr :5000
```

Убейте процесс:
```bash
taskkill /PID [номер_процесса] /F
```

Или измените порт в `api_server.py` (последняя строка):
```python
app.run(host='0.0.0.0', port=5001, debug=False)
```

И в конфиге плагина:
```yaml
detection:
  endpoint: "http://localhost:5001"
```

### Ошибка: "Model not loaded"

Модель не обучена. Запустите:
```bash
python train_model.py
```

Или:
```bash
retrain.bat
```

## Правильный порядок запуска

1. ✅ **Сначала** запустите API сервер:
   ```bash
   python api_server.py
   ```

2. ✅ **Потом** запустите Minecraft сервер

3. ✅ Проверьте логи - должно быть "AI проверка включена!"

---

## Альтернатива: Включить через команду

Если не хотите перезапускать сервер:

1. Измените конфиг как описано выше
2. В игре введите:
   ```
   /smartai reload
   ```

Но это может не сработать если API сервер не был запущен при старте плагина. Лучше перезапустить сервер.

---

## Проверка работы AI

После включения:

1. Зайдите в игру
2. Введите:
   ```
   /smartai prob [ваш_ник]
   ```

3. Побейте моба несколько раз

4. В actionbar должна появиться информация о вероятности

5. Над вашей головой должна появиться голограмма с последними 5 ударами

---

## Если все равно не работает

Проверьте:

1. ✅ API сервер запущен (`python api_server.py`)
2. ✅ Модель обучена (`smartai_model.pkl` существует)
3. ✅ В конфиге `detection.enabled: true`
4. ✅ Endpoint правильный: `http://localhost:5000`
5. ✅ Порт 5000 не заблокирован файрволом
6. ✅ Сервер перезапущен после изменения конфига

Если проблема остается, покажите полные логи сервера при запуске.
