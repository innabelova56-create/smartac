# 🚀 Быстрый старт SmartAI

## TL;DR - Где найти AI?

**Ответ: Создай сам за 30 минут!**

```bash
# 1. Собери данные на сервере
/smartai start legit  # играй 50+ боев
/smartai stop
/smartai start cheat  # включи читы, играй 50+ боев
/smartai stop

# 2. Обучи модель
pip install -r requirements.txt
python train_model.py

# 3. Запусти API
python api_server.py

# 4. Готово!
curl http://localhost:5000/health
```

## Что это?

SmartAI - это AI античит для Minecraft, который:
- ✅ Собирает данные ротаций игроков
- ✅ Обучает ML модель отличать легитов от читеров
- ✅ Предоставляет API для детекции в реальном времени

## Файлы проекта

### Плагин (Java)
- `src/main/java/com/smart/smartAI/` - исходники плагина
- `build.gradle` - конфигурация сборки
- `plugin.yml` - метаданные плагина

### AI модели (Python)
- `train_model.py` - обучение Random Forest (быстро, просто)
- `train_lstm_model.py` - обучение LSTM (медленно, точнее)
- `api_server.py` - API для Random Forest
- `api_server_lstm.py` - API для LSTM
- `test_api.py` - тестирование API

### Документация
- `README.md` - общее описание
- `AI_SETUP.md` - **ГЛАВНАЯ ИНСТРУКЦИЯ** по созданию AI
- `USAGE.md` - подробное руководство
- `CHECKLIST.md` - чеклист для проверки
- `QUICK_START.md` - этот файл

## Минимальные требования

### Для плагина
- Java 21
- Paper 1.21.4
- 1GB RAM

### Для AI
- Python 3.8+
- 2GB RAM
- 1GB свободного места

### Для данных
- Минимум 50 боев легитных игроков
- Минимум 50 боев с читами
- Рекомендуется 200+ каждого типа

## Пошаговая инструкция

### Шаг 1: Установи плагин (5 минут)
```bash
./gradlew build
cp build/libs/SmartAI-1.0.jar /path/to/server/plugins/
# Перезапусти сервер
```

### Шаг 2: Собери данные (30-60 минут)
```bash
# На сервере
/smartai start legit
# Играй в PvP, проведи 50+ боев
/smartai stop

/smartai start cheat
# Включи KillAura, проведи 50+ боев
/smartai stop
```

Данные сохранятся в `plugins/SmartAI/datasets/*.csv`

### Шаг 3: Обучи AI (5-10 минут)
```bash
# Скопируй CSV файлы на компьютер с Python
# Установи зависимости
pip install pandas scikit-learn numpy joblib flask

# Обучи модель
python train_model.py
```

Получишь файл `smartai_model.pkl` - это твоя AI модель!

### Шаг 4: Запусти API (1 минута)
```bash
python api_server.py
```

API работает на `http://localhost:5000`

### Шаг 5: Проверь (1 минута)
```bash
python test_api.py
```

Должен показать:
```
✅ Сервер работает
✅ Предсказание получено
```

3. Регулярно переобучай модель

## Частые вопросы

**Q: Где скачать готовую AI модель?**
A: Нигде. Каждый сервер уникален, модель нужно обучать на своих данных.

**Q: Сколько нужно данных?**
A: Минимум 50 боев каждого типа. Рекомендуется 200+. Идеально 1000+.

**Q: Какая точность будет?**
A: 
- 50 боев → ~70%
- 200 боев → ~85%
- 1000+ боев → ~95%+

**Q: Random Forest или LSTM?**
A: 
- Random Forest - если мало данных или нет GPU
- LSTM - если много данных и нужна максимальная точность

**Q: Как долго обучается?**
A:
- Random Forest: 1-5 минут
- LSTM: 10-30 минут

**Q: Нужен ли GPU?**
A: Нет для Random Forest. Желательно для LSTM (но не обязательно).

**Q: Можно ли использовать на продакшн сервере?**
A: Сначала протестируй на тестовом сервере. Собери статистику false positives.

**Q: Как обновлять модель?**
A: Собирай новые данные, переобучай модель, заменяй файл .pkl/.h5
